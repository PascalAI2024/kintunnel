# Brand

## Name

The project name is **KinTunnel**.

It signals the intended audience: people you trust, not anonymous customers or a corporate fleet. It also says exactly what the software does: builds a tunnel through your own server.

## Tagline

Self-hosted WireGuard access for people you trust.

## Short Description

KinTunnel is a Docker-native WireGuard VPN manager for friends, family, and small trusted teams.

## Positioning

- Personal and small-group VPN management.
- One VPS as the first supported shape.
- Clear peer lifecycle workflows.
- Conservative security defaults.
- Dry-run first until host networking automation is production hardened.
