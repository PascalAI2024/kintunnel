# Support

KinTunnel is early open-source software. The dry-run manager is usable for evaluation; production host-networking automation is still being hardened.

## Questions and Help

- Start with [README.md](README.md).
- Use [docs/quick-start.md](docs/quick-start.md) for local and Docker evaluation.
- Use [docs/troubleshooting/index.md](docs/troubleshooting/index.md) for common operator checks.
- Open a GitHub issue for reproducible bugs or focused feature requests.

## Security Reports

Do not open public issues for vulnerabilities, leaked keys, authentication bypasses, or deployment-sensitive findings. Use [GitHub private vulnerability reporting](https://github.com/PascalAI2024/kintunnel/security/advisories/new) and follow [SECURITY.md](SECURITY.md).

## Scope

This project does not provide hosted VPN service support, legal advice, abuse handling, or provider-specific firewall consulting. The VPS operator owns those responsibilities. Annoying, but still true.
